# 项目说明文档

## 项目概述

这是一个文档转换系统的说明文档，演示了Markdown格式的基本用法。

### 系统功能

该系统支持以下文档格式的转换：

1. **Microsoft Word文档** (.docx, .doc)
   - 支持文本内容提取
   - 保持基本格式结构
   - 处理嵌入的图片

2. **PDF文档** (.pdf)
   - 文本内容提取
   - 自动段落识别
   - 标题层级检测

3. **HTML文档** (.html, .htm)
   - 完整的HTML到Markdown转换
   - 保持语义结构
   - 支持表格、列表等元素

### 技术特性

- **实时监控**: 自动检测文件变化
- **增量同步**: 只处理修改过的文件
- **Web界面**: 直观的管理界面
- **实时日志**: 详细的转换过程记录

### 使用示例

```javascript
// 启动服务器
npm start

// 开发模式
npm run dev
```

### 配置说明

系统默认配置：
- 监控目录: `./data`
- 输出目录: `./data_markdown`
- 服务端口: `3000`

> **注意**: 请确保源目录具有读取权限，输出目录具有写入权限。
